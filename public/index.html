CSGO API

An unofficial JSON API for CS:GO

List weapons

This list includes knives and gloves. This will be useful in the next endpoint where the skins are grouped by weapons ids.

GET https://bymykel.me/CSGO-API/api/weapons.json

        "id": 1,
